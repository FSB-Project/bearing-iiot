# Mô hình Học máy (ML Models)

Các thuật toán và mô hình được sử dụng để dự báo tình trạng vòng bi.

## Thuật toán
- **Anomaly Detection**: Sử dụng Isolation Forest hoặc Autoencoders.
- **Remaining Useful Life (RUL)**: Sử dụng LSTM hoặc GRU.

## Dữ liệu huấn luyện
- Dữ liệu lịch sử từ phòng thí nghiệm và thực tế.
