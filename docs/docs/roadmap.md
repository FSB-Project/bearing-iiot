# Lộ trình phát triển (Roadmap)

Kế hoạch phát triển các tính năng trong tương lai.

## Giai đoạn 1 (Hiện tại)
- Hoàn thiện hệ thống thu thập dữ liệu cơ bản.
- Xây dựng dashboard hiển thị thời gian thực.

## Giai đoạn 2 (Sắp tới)
- Tích hợp mô hình dự báo hư hỏng (PdM).
- Ứng dụng AI/ML cho phân tích rung động chuyên sâu.

## Giai đoạn 3
- Hệ thống hóa cho nhiều loại thiết bị quay khác.
